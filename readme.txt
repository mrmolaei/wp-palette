A WordPress plugin to add custom color palette to Gutenberg block editor.

== Description ==

This plugin will help you add a custom color palette to block editor to use in you blocks.

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.

== Frequently Asked Questions ==

= How do I use this plugin? =

After installing the plugin, go to Appearance -> WP Palette and there you can add your colors.

= How to uninstall the plugin? =

Simply deactivate and delete the plugin.
